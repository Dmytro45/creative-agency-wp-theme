# Creative Agency Portfolio - WordPress & Bootstrap 5 Custom Theme

This is a lightweight, fully responsive, and SEO-friendly custom WordPress theme built from scratch using **Bootstrap 5** and native WordPress core functions. 

## 🚀 Features
- Custom Post Type (CPT) for Portfolio projects.
- Fully responsive layout utilizing Bootstrap 5 grid, cards, and navigation.
- Dynamic homepage showcasing the latest 3 projects using custom `WP_Query`.
- Separate dedicated templates for single project pages (`single-portfolio.php`) and project archives (`archive-portfolio.php`).
- Optimized for speed with zero heavy page-builders.

## 🛠️ Built With
- WordPress Core (PHP)
- Bootstrap 5
- Semantic HTML5 & Custom CSS3
